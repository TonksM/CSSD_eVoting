(inner) isNotAWP Checks if the current user is an associated user of a proxy

This function checks if the Voter is present within any other voter's proxyFor_ field and if not directs them along the normal route. If they have are an associated voter for a proxy voter, they will be redirected to a short message state who is voting on their behalf
